About Kompass

 

If we get to a new tourist place, the google map may be used to get the information about it. But some of it is hard to understand and some have less information. So we are introducing a new generation application named ‘Kompass’. Kompass makes exploring the world easier by organizing the essential information about a tourist spot in one place.

The existing System will not provide each and every minute details about a single place. ’Kompass’ will provide photos, details, facilities, fee structure, ratings, reviews about a place and also will get suggestions based on what’s nearby.
